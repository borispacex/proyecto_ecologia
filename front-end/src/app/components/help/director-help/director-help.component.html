¿Cómo administrar proyectos de investigación?

Para poder acceder como administrador usted debe tener acceso como director, si tiene mas roles en el sistema debe elegir ingresar modo DIRECTOR, una vez hecho este paso.
El primer menu que usted podra acceder sera el menu principal, donde tendra las siguinetes opciones:
  • Listar Proyectos
  • Listar Investigadores
  • Generar Reportes
Tiene dos formas de acceder un proyecto como director:
  • Listar Proyectos: Ahi podra observar todos los proyectos de investigacion.
  • Listar Investigadores: Ahi podra observar todos los investigadores y con ello cada investigador tendra su lista de proyectos.
Una vez que estes en la pagina del proyecto de investigación elegido, usted podra observar los datos del proyectos de investigación como datos, documentos etc.
Debe elegir el boton Realizar seguimiento, donde usted DIRECTOR debe llenar los siguientes datos:
  • Tipo de seguimiento (Inicio, Ejecucion o Finalización)
  • Revisión
  • Observaciones
  • Proceso
  • Subir archivos importantes para el seguimiento
Podra observar una lista de seguimientos, donde podras editar o eliminar el seguimiento si deseas.

¿Cómo administrar investigadores?

Para poder acceder como administrador usted debe tener acceso como director, si tiene mas roles en el sistema debe elegir ingresar modo DIRECTOR, una vez hecho este paso.
El primer menu que usted podra acceder sera el menu principal, donde tendra las siguinetes opciones:
  • Listar Proyectos
  • Listar Investigadores
  • Generar Reportes
Debe elegir la opción elegir Listar Investigadores, donde usted observara una gran lista de investigadores.
Podra observar los campos de investigador:.
  • Nombre completo
  • Cedula de identidad
  • Foto
  • Correo electronico
  • Tipo de investigador
  • Grado academico
  • Lista de proyectos de investigación

¿Cómo generar reportes?

Para poder acceder como administrador usted debe tener acceso como director, si tiene mas roles en el sistema debe elegir ingresar modo DIRECTOR, una vez hecho este paso.
El primer menu que usted podra acceder sera el menu principal, donde tendra las siguinetes opciones:
  • Listar Proyectos
  • Listar Investigadores
  • Generar Reportes
Debe elegir la opción elegir Generar Reportes, donde usted observara un gran menu de opciones para generar reportes con muchos parametros.
Podra generar reportes de muchas maneras y observar en la tabla como sera exportado.
Podra generar reportes de muchas maneras y exportarlos en muchos formatos:
  • Impresión
  • Excel
  • CSV
  • JSON
  • TEXT
