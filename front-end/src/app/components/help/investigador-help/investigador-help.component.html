¿Cómo administrar proyectos de investigación?

Para poder acceder como administrador usted debe tener acceso como investigador, si tiene mas roles en el sistema debe elegir ingresar modo INVESTIGADOR, una vez hecho este paso.
El primer menu que usted podra acceder sera el menu principal, donde tendra las siguinetes opciones:
  • Listar Proyectos
  • Perfil de Investigador
Una vez se encuentre en el Menu principal usted debe elegir la opción Listar Proyectos, una vez se encuentre ahi, podra observar los proyectos de investigacion que forma parte, tendra dos formas:
  • Proyectos de investigación en el que es coordinador (ICONO OJO)
  • Proyectos de investigación en el que es investigador (ICONO LAPIZ)
Como ya habiamos mencionado existen dos formas de acceder a un proyecto como investigador, cuando usted es coordinador o investigador. Coordinador Tendra las opciones de agregar los siguientes campos
  • Realizar publicación
  • Basica tecnica (Informacion del proyecto)
  • Lugar de desarrollo
  • Difusiones
    • Cursos, Seminarios y Talleres
    • Eventos cientificos
    • Notas de prensa
    • Exposiciones y conferencias

¿Cómo dar de concluido el proyecto?

Para dar de concluido el proyecto de investigación y hacer cierre de proyecto debe comunicarse con el administrador para notificarle cierre de proyecto, el administrador subira un informe final, elaborado por el coordinador.
El Director debe hacer una revisión, es decir un seguimiento de etapa final, para verificar todo el avance y asi finalmente dejar el proceso de proyecto como 100%.
El administrador debe ingresar a su proyecto de investigación y subir un archivo final elaborado por el coordinador.

¿Cómo realizar una publicación?

Unicamente el investigador Coordinador podra realizar publicaciones, si usted no es coordinador de algun proyecto de investigación no sera capaz de hacer publicaciónes.
Para realizar la publicación debe dirigirse a su lista de proyectos y seleccionar el proyecto de investigación que desea realizar la publicación.
Una vez elegido el proyecto de investigación, usted debe seleccionar el boton Realizar Publicación. Nota: Debe realizar subir un archivo no mayor a 100MB
Estos campos son muy importantes para el llenado de la publicación.
  • Tipo de publicación (Articulo, Libro, Poster, Metodo, Tesis, Investigación, Propuesta, etc)
  • Titulo
  • Autores
  • Fecha
  • Cita bibliografica
  • Campo para subir archivos (PDF o IMAGEN)
Usted podra editar la publicación, buscando en su listado de publicaciones y seleccionando el boton de Editar.
El usuario Director es el unico que puede realizar un comentario a la publicación, para eso el director debe dirigirse al listado de proyectos o los proyectos de algun investigador y seleccionar alguno, donde podra realizar el comentario observando el listado de publicaciones y tendra acceso a realizar un comentario.

¿Cómo realizar difusiones?

Unicamente el usuario Investigador de tipo coordinador podra realizar difusiones del proyectos de investigación, para eso debe dirigirse al listado de proyectos y seleccionar el proyecto de investigación, despues seleccionarlo y tendra muchas opciones.
Una vez seleccionado el proyecto de investigación usted debe realizar la opcion Difusiones donde sera capaz de realizar distintos tipos de difusion:
  • Cursos, Seminarios y Talleres
  • Eventos cientificos
  • Notas de prensa
  • Exposiciones o conferencias
Puede seleccionar cualquiera de ellos para realizar difusión.
Una vez elegido el tipo de difusión, los campos principales que tenemos son:
  • Titulo
  • Objetivo
  • Lugar de realización
  • Fecha
  • Resumen
  • Campo para subir archivos
Una vez llenado los campos que son necesarios para la difusion, se subira al sistema. En la parte de abajo podra observar el listado de todas las difusiones que realizo.
