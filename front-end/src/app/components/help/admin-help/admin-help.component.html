¿Cómo administrar cuentas?

Para poder acceder como administrador usted debe tener acceso como administrador, si tiene mas roles en el sistema debe elegir ingresar modo ADMIN, una vez hecho este paso.
El primer menu que usted podra acceder sera el menu principal, donde tendra las siguinetes opciones:
  • Listar Fotografias
  • Listar Usuarios
  • Listar Proyectos
Una vez se encuentre en el Menu principal usted debe elegir la opción Listar Usuarios, una vez este ahi tendra las opciones de AGREGAR USUARIO y observar el listado de todos los usuarios del sistema, donde podra observar datos de cada uno y editar algunos campos del usuario.
Podra observar un listado de usuarios donde podra listarlos por Activos, Inactivos o Ambos. Tambien podra Listar por roles Administrador, Director e Investigador.
Tiene la opcion de agregar usuarios con muchos roles (Admin, Director e Investigador), tambien la opción de quitar algun rol.

¿Cómo crear proyectos de investigación?

Para poder acceder como administrador usted debe tener acceso como administrador, si tiene mas roles en el sistema debe elegir ingresar modo ADMIN, una vez hecho este paso.
El primer menu que usted podra acceder sera el menu principal, donde tendra las siguinetes opciones:
  • Listar Fotografias
  • Listar Usuarios
  • Listar Proyectos
Una vez se encuentre en el Menu principal usted debe elegir la opción Listar Proyectos, una vez este ahi tendra las opciones de AGREGAR PROYECTO y observar el listado de todos los proyectos de investigación del sistema, donde podra observar datos de cada uno y editar algunos campos del usuario.
Para agregar el proyecto de grado tiene que tener los siguientes campos:
  • Titulo de proyecto
  • Fecha inicio (dd/mm/yyyy)
  • Fecha fin (dd/mm/yyyy/)
  • Nombre del coordinador (Debe estar registrado en el sistema)
  • Nombre de investigadores (Deben estar registrados en el sistema)
  • Archivos principales (Inextenso, Formulario o Resolución de proyecto)

¿Cómo administrar proyectos de investigación?

Para poder acceder como administrador usted debe tener acceso como administrador, si tiene mas roles en el sistema debe elegir ingresar modo ADMIN, una vez hecho este paso.
El primer menu que usted podra acceder sera el menu principal, donde tendra las siguinetes opciones:
  • Listar Fotografias
  • Listar Usuarios
  • Listar Proyectos
Una vez se encuentre en el Menu principal usted debe elegir la opción Listar Proyectos, una vez este ahi tendra las opciones de AGREGAR PROYECTO y observar el listado de todos los proyectos de investigación del sistema, donde podra observar datos de cada uno y editar algunos campos del usuario.
Una vez que este en el listado de proyectos, podra observar los siguientes campos:
  • Estado
  • Titulo
  • Proceso
  • Grupo
  • Acciones (Editar, Eliminar)
Donde podras agregar proyectos de investigación, Editar el proyecto de investigación cambiando datos del proyecto.
Una vez este en el listado de proyectos de investigación, podra elegir algun proyecto de investigación y podra agregar mas campos a ese proyecto como por ejemplos documentos principales del proyectos.
Una vez este en el menu o pagina del proyecto de investigación seleccionado, usted podra agregar los siguientes campos:
  • Subi archivos (Tipo de archivo: Principal u Otros)
  • Permiso (Tipo de nota: DGB, SERNAP, CITES, Bioetica, Cierre de Proyecto, Otros) (Tipo permiso: Solicitud u Otorgación)
  • Suscripción convenio
  • Personal contratado
  • Cierre de proyecto
En la parte de abajo podras observar el listado de todos estas opciones.
