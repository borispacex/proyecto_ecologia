¿Cómo acceder a una cuenta?

Cualquier usuario que quiera acceder a una cuenta al SISTEMA DE SEGUIMIENTO Y CONTROL DE PROYECTOS INVESTIGACION debe apersonarse a instalaciones de la dirección del Instituto de Ecologia, ubicado en Campus Universitario: Calle 27 Cota - Cota, Zona sur. Posteriormente debe apersonarse con el administrador del sistema con sus datos personales:
  • Fotocopia Cedula de identidad (Nombre completo, Mencion, CI)
  • Correo electronico
  • Tipo de usuario con el que desea acceder (Investigador Titular, Investigador asociado con proyecto o Investigador por invitación)
¿Podria hacer este proceso de manera online?
Enviando al correo electronico a: dir.ie.umsa@gmail.com los datos ya mencionados.
Una vez que el administrador haya agregado sus datos personales al Sistema, el usuario recibira un correo electronico dandole la bienvenida al sistema, notificandole unas recomendaciones que debe hacer por seguridad una vez inicie sesion por primera vez.
La primera vez que ingrese al sistema el usuario debe cambiar la contraseña por seguridad y se recomienda que el usuario actualice sus datos personales para tener mejor comunicación con el Instituto de Ecologia. Si el usuario quiere cambiar de correo electronico debe comunicarse con el administrador del sistema, porque el correo electronico es una parte fundamentalmente para el manejo y comunicacion con el sistema, por ejemplo con el correo electronico usted podra restaurar la contraseña en caso de que se le olvido.

¿Cómo dar de baja mi cuenta?

Cualquier usuario que quiera dar de baja a una cuenta, debe apersonarse a instalaciones de la dirección del Instituto de Ecologia, ubicado en Campus Universitario: Calle 27 Cota - Cota, Zona sur. Posteriormente debe apersonarse con el administrador del sistema con sus datos personales:
  • Fotocopia Cedula de identidad (Nombre completo, Mencion, CI)
¿Podria hacer este proceso de manera online?
Enviando al correo electronico a: dir.ie.umsa@gmail.com los datos ya mencionados. Nota: El usuario no sera borrado del sistema, unicamente ya no tendra acceso. Todas sus publicaciónes estaran guardadas en el sistema.
El administrador con los datos del usuario que desea ser dado de baja, realizara la inhabilitación del usuario.

¿Qué hago si olvide mi contraseña?

El usuario que olvido su contraseña debe dirigirle a la pagina principal de inicio de sesión, elegir la opción ¿Olvido su contraseña?, posteriormente debe ingresar el correo electronico que esta registrado en el sistema.
En caso de que no tenga acceso a ese correo electronico, debe comunicarse con el administrador del sistema para hacer la posterior actualización del correo electronico.
Una vez que que introduzca el correo electronico, click RESTABLECER CONTRASEÑA. Recibira un correo electronico con un codigo y enlace para reestablecer la contraseña.
Una vez haya recibido el correo electronico, debera ingresar al enlace que recibio del sistema para restablecer la contraseña.
Debe ingresar al enlace que recibio en el correo electronico, donde solo tiene tiempo limite para hacer el restablecimiento de contraseña, posteriormente el tiempo expire ya no tendra permitido acceder al enlace.
Una vez ingresado al enlace, usted tendra la opcion de cambiar la contraseña debe ingresar la contraseña siempre teniendo en cuenta MAY MINUSCULAS y SIMBOLOS. Una vez sea validado cuando ingrese nuevamente al sistema podra acceder con la nueva contraseña.
